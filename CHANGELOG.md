# Change Log

All notable changes to the "vscode-zotero-cite" extension will be documented in this file.

## [0.1.0] - TBD

### Added
- Initial release
- Automatic citation insertion via `\zoteroCite` trigger command
- Integration with Zotero's native CAYW picker
- BibTeX entry fetching from Better BibTeX plugin
- Automatic .bib file management with duplicate detection
- Configuration options for default .bib file and auto-append behavior
- Cross-platform support (Windows, macOS, Linux)
